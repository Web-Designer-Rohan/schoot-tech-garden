How-To & Tutorials

Step-by-step guides to help you with common school tech tasks. Learn at your own pace!

How to Make a Bootable USB Drive

1

Download the ISO file

Get the operating system ISO file you want to install (Windows, Linux, etc.) from the official source.

2

Download Rufus

Download the free Rufus tool (for Windows) or Balena Etcher (for Mac/Linux).

3

Insert your USB drive

Use a USB with at least 8GB capacity. All data will be erased, so back up important files first.

4

Run Rufus and select your USB

Open Rufus, select your USB from the device dropdown menu.

5

Select the ISO file

Click "SELECT" and browse to your downloaded ISO file.

6

Start the process

Click "START" and wait for the process to complete (may take 10-30 minutes).

Tip: On some computers, you may need to change the boot order in BIOS to boot from USB.

How to Install Windows (On Supported & Unsupported Computers)

Installing Windows Normally

1

Create installation media

Use the Media Creation Tool from Microsoft to create a bootable USB.

2

Boot from USB

Restart your computer and boot from the USB (usually by pressing F12, F2, or DEL during startup).

3

Follow installation prompts

Select language, edition, and installation type (Custom for clean install).

4

Complete setup

Follow the on-screen instructions to complete Windows setup.

Installing/Upgrading on Unsupported Computers

Warning: These methods may not work on all hardware and could cause instability.

1

Modify the ISO file

Use tools like MediaCreationTool.bat to create an ISO without hardware checks.

2

Edit registry during setup

At the installation screen, press Shift+F10 to open command prompt and run registry edits to bypass checks.

3

Use alternative installation methods

Consider lightweight versions of Windows or alternative operating systems for very old hardware.

Tip: Always back up your data before attempting to install on unsupported hardware.
